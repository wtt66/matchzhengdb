# 第四部分 假设检验与结构方程模型结果分析

## 4.1 测量模型修正与检验

基于第四部分量表数据，本文首先对购买意愿影响因素模型进行验证性因子分析。初始测量模型包含文化价值感知（CVP）、产品知识（PK）、购买便利性（PC）、经济可及性（EA）、感知风险（PR）、产品涉入度（PI）、先验知识（PKN）和购买意愿（BI）8个潜变量，共27个观测题项。考虑到题项之间可能存在载荷偏弱、区分效度不足与误差方差偏高等问题，本文在完整保留审计过程的基础上，对测量模型进行了透明修正。

题项修正遵循以下原则：其一，标准化载荷低于0.35的题项被界定为弱题项；其二，当题项在所属构念上的载荷与最大跨构念负荷之差低于0.08时，认为其区分效度不足；其三，当题项载荷低于0.40且残差方差占比高于0.85时，将其视为高残差弱题项。依据上述标准，最终剔除了PC1、EA2、PR3与PI3四个题项，并在保证每个构念至少保留2个题项的前提下形成优化后的测量模型。修正后模型共保留23个观测题项。

优化后测量模型的整体拟合结果良好：$\chi^2=334.515$，$df=202$，CFI=0.942，TLI=0.927，RMSEA=0.033，SRMR=0.043。与基线模型相比，CFI由0.882提升至0.942，TLI由0.860提升至0.927，RMSEA由0.042下降至0.033，SRMR由0.055下降至0.043，说明通过题项修正后，模型的拟合质量得到显著改善。

从信效度结果看，优化后各潜变量的复合信度（CR）介于0.335至0.605之间，平均方差提取量（AVE）介于0.211至0.338之间。其中，产品知识（PK）和先验知识（PKN）的测量稳定性相对较好，CR分别为0.605和0.603；购买便利性（PC）、经济可及性（EA）与感知风险（PR）的CR和AVE仍相对偏低，说明原始问卷在这些构念上的题项设计仍有提升空间。因此，本文认为优化后的测量模型已足以支撑后续结构路径解释，但相关结论仍应在“可接受但并非完美量表”的前提下进行审慎讨论。

## 4.2 结构模型比较与最终模型确定

在结构模型部分，本文分别比较了三类模型：基线全模型、优化全模型和最终精简模型。基线全模型保留全部主效应和调节效应路径；优化全模型在修正测量模型后继续保留全部结构路径；最终精简模型则仅保留在主效应检验中表现稳定、显著且具有明确理论意义的核心路径。

模型比较结果显示，基线全模型的调整后决定系数为0.403，优化全模型为0.396，最终精简模型为0.387。尽管精简模型的解释率略低于全模型，但其5折交叉验证均方根误差（CV RMSE）为0.784，和优化全模型的0.787非常接近，同时BIC由1456.950下降至1433.263，说明在控制模型复杂度后，精简模型具有更好的简约性与更稳定的泛化表现。基于“解释性、稳健性与图示清晰性”的综合考量，本文将最终精简模型作为正式论文的主报告模型，并将全模型结果保留为补充性检验。

最终精简模型保留的结构路径包括：文化价值感知（CVP）→购买意愿（BI）、产品知识（PK）→购买意愿（BI）、经济可及性（EA）→购买意愿（BI）、产品涉入度（PI）→购买意愿（BI）以及先验知识（PKN）→购买意愿（BI）。购买便利性（PC）、感知风险（PR）、非遗融入接受度（ACCEPT）以及三个调节效应项在优化后的全模型中均未表现出稳定显著效应，因此未纳入最终主模型。

## 4.3 假设检验结果

依据理论框架，本文对以下假设进行检验：H1认为文化价值感知正向影响购买意愿；H2认为产品知识正向影响购买意愿；H3认为购买便利性正向影响购买意愿；H4认为经济可及性正向影响购买意愿；H5认为感知风险负向影响购买意愿；H6认为产品涉入度正向影响购买意愿；H7认为先验知识正向影响购买意愿；H8a认为产品涉入度正向调节文化价值感知与购买意愿的关系；H8b认为产品涉入度正向调节购买便利性与购买意愿的关系；H9认为先验知识削弱感知风险对购买意愿的负向作用。

在最终主模型中，H1、H2、H4、H6和H7得到支持，H3和H5未得到支持；在优化后的全模型中，H8a、H8b和H9亦均未获得统计支持。具体检验结果见表4-1。

## 4.4 结构路径结果解释

最终模型结果表明，文化价值感知、产品知识、经济可及性、产品涉入度和先验知识均对购买意愿产生显著正向影响。其中，文化价值感知（$\beta=0.228$）和先验知识（$\beta=0.226$）的影响最强，说明对于福州本土国潮香氛而言，消费者是否能够感知到文化价值、以及是否具备相应的香氛识别与判断基础，是推动购买意愿形成的两个核心前提。这一结果意味着，文化叙事并非单纯的符号装饰，而是在消费者认知层面真正构成购买决策的重要依据。

产品知识（$\beta=0.200$）和产品涉入度（$\beta=0.156$）同样表现为显著正向驱动因素，表明消费者对产品香型、工艺、品质和使用方式了解越充分，对香氛产品投入的关注程度越高，其购买福州本土国潮香氛的意愿也越强。换言之，本地品牌若希望实现从“文化认知”到“购买转化”的有效跃迁，仅依赖文化标签本身并不足够，还需要通过产品说明、场景教育和品牌传播增强消费者的知识获得与涉入体验。

经济可及性（$\beta=0.108$）在最终模型中也达到显著水平，虽然其效应强度弱于文化价值感知、产品知识和先验知识，但仍说明价格可接受性与消费预算匹配是影响购买意愿的重要现实条件。这一发现提示，福州本土国潮香氛在构建文化溢价时需要保持一定边界，即文化表达应服务于价值提升，而不能脱离消费者对价格合理性的基本判断。

值得注意的是，购买便利性、感知风险和非遗融入接受度在优化后的全模型中均未呈现稳定显著效应。这表明，在当前样本中，消费者的购买意愿并不主要受“渠道是否方便”或“风险担忧是否强烈”直接驱动，而更取决于其是否真正理解并认同产品的文化价值、是否具备基本知识储备，以及是否对香氛消费本身具有较高涉入水平。同时，三个调节项均未达到显著水平，说明产品涉入度和先验知识更多表现为直接作用变量，而非通过显著改变其他路径强度来间接影响购买意愿。

## 4.5 结果讨论

综合来看，福州本土国潮香氛的购买意愿形成机制并不是单一价格机制或单一渠道机制，而是一个以“文化价值感知—知识理解—涉入体验”为核心、并辅以“经济可及性”支撑的多因素综合过程。换言之，消费者并不是因为单纯听说过相关文化概念而愿意购买，也不是因为简单接触到某个购买渠道就会形成强烈意愿，而是在能够感知文化表达、理解产品差异并形成较高涉入后，才更可能转化为真实的购买倾向。

这一结果对本土品牌运营具有明确启示。首先，应强化文化价值的可感知化表达，把福州茉莉花窨制、冷凝合香及地域文化元素从包装叙事进一步转化为可体验、可解释、可识别的产品价值。其次，应重视知识传播与消费教育，通过门店体验、内容种草和产品信息可视化提升消费者对香型结构、工艺差异与使用场景的理解。再次，在文化溢价之外仍需兼顾经济可及性，避免价格体系与消费者感知价值脱节。最后，对于渠道布局和风险管理，虽然其在本研究中未进入最终主模型，但并不意味着它们不重要，而是说明这些因素更可能属于基础保障条件，而非当前阶段的核心解释变量。

总体而言，本文的结构方程模型结果进一步证明，福州本土国潮香氛市场当前面临的关键问题不在于“文化元素缺失”，而在于文化价值尚未被充分转化为消费者能够稳定感知和接受的产品认知体系。只有当文化表达、知识理解和产品体验真正形成协同，购买意愿才可能实现更稳健的提升。
