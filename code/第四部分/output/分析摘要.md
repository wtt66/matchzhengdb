# 购买意愿影响因素：结构方程模型分析

本部分围绕文化价值感知、产品认知、购买便利性、经济可及性、风险感知、涉入度、先验知识与购买意愿等潜变量展开，适合进行量表信度检验和路径回归分析。 本轮输出重点重构为测量模型、结构路径、稳健性诊断和补充判别效度图。

## 关键发现

- 第四部分量表共 596 个有效样本、23 个观测题项、5 个主报告采用的潜变量，是整份问卷中最适合开展结构方程模型分析的章节。
- CFA 拟合结果为 CFI=0.928、TLI=0.917、RMSEA=0.035、SRMR=0.048，说明模型达到可解释且较稳定的拟合水平。
- 最终精简结构模型的调整后 R² 为 0.387，5折交叉验证 RMSE 为 0.783，说明模型在简化后仍保持稳定解释力。
- 主路径中显著正向作用最强的因素依次集中在 购买准备度, 先验知识, 文化价值感知。
- 当前没有稳定的显著负向路径。
- 精简过程中删减的路径包括 感知风险, 非遗融入接受度。
- 信效度方面，复合信度最高的是 购买准备度（CR=0.772），AVE 相对较弱的是 文化价值感知（AVE=0.240）。
- 分箱校准的平均绝对误差为 0.184，说明结构模型在高低购买意愿群体之间具有较稳定的区分能力。

## 模型说明

本部分采用两步法结构方程模型思路：先对当前保留题项进行验证性因子分析（CFA），再依据标准化载荷构建潜变量加权得分，并估计购买意愿的结构路径。主报告版本进一步采用合并高相关构念后的简约结构，用于缓解判别效度不足问题。

## 结果解读

综合来看，购买意愿的核心驱动主要来自 购买准备度, 先验知识, 文化价值感知。其中，购买准备度 对购买意愿的影响最强。测量模型层面，大多数题项在所属潜变量上的载荷明显高于跨构念相关，说明第四部分量表可以支撑后续结构路径解释。合并高相关构念后，主报告中的路径解释更偏向‘文化价值 + 购买准备 + 先验知识’的三轴框架。因此，在论文写作中可以把第四部分作为整份问卷的核心 SEM 章节，前面几部分更多承担认知、行为、场景和细分的背景解释作用。
