## 3. 模型说明

- 最终路径：文化价值感知, 产品知识, 经济可及性, 产品涉入度, 先验知识
- 判别效度通过率：12.5%
- 最小判别效度边际：-0.491
- 平均 CR / AVE：0.506 / 0.276

### 备选A：购买准备度合并模型

将产品知识、购买便利性、经济可及性与产品涉入度合并为“购买准备度”，用于压缩高相关的决策准备类构念。

- 最小判别效度边际：-0.380
- 平均 CR / AVE：0.589 / 0.276

将购买便利性与经济可及性合并为“交易可得性”，将产品知识、产品涉入度与先验知识合并为“认知卷入基础”。

## 4. 推荐结论

- 推荐模型：备选A：购买准备度合并模型
