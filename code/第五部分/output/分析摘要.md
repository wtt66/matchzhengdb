# 消费痛点与改进建议分析

本部分以问卷显性痛点、平衡后的公开网页语料与 BERTopic 主题结构为三类核心证据，并以问卷开放题作为本地化语义补充，形成面向论文写作的消费痛点识别框架。图表按重构版 5.1-5.7 重新组织，重点突出三源验证、来源差异与显隐性映射关系。

## 关键发现

- 问卷显性痛点中，品质/留香是最突出的结构性障碍；改进诉求则优先指向文化融合。
- 第五部分以问卷显性痛点、平衡后的公开网页语料与 BERTopic 主题结构为三类核心证据；本轮新增 23 个公开网页页面，清洗后形成 648 条 BERTopic 可建模文本，其中问卷开放题 351 条、公开网页语料 297 条。
- 三源分面条形图显示，平衡后的公开网页语料把“品质/留香”与“香型/气候适配”的体验表达显著放大，说明外部语料主要补足了使用后反馈，而问卷继续保留了伴手礼与在地文化的本土议题。
- BERTopic 隐性主题中，问卷开放题最集中的主题是“福州本土香氛开发”，公开网页语料最集中的主题是“通用香水体验评价”，表明两类语料分别代表‘在地化期待’与‘通用体验评价’两种不同证据层。
- 显隐性映射矩阵中，最强耦合关系出现在“伴手礼适配—礼赠定制与产品延展”，说明显性痛点与隐性主题已经能形成交叉验证，而不再是彼此孤立的两套结论。
- 机会排序图显示，当前最值得优先投入的方向是“文化融合”，它同时兼具较高痛点强度与改进诉求强度。
- 新增的中式传统香炉词云图综合了问卷、开放题与公开网页资料，仅保留名词或名词性短语；当前权重最高的词项包括 香薰, 香水, 香氛, 福州, 味道, 品牌。

## 研究设计与证据整合

第五部分以问卷显性痛点为显性证据，以平衡后的公开网页语料补足外部体验语言，并将问卷开放题与公开网页文本共同纳入 BERTopic 主题建模，形成三类证据链。
其中，结构化问卷负责回答“哪些问题最常被明确提及”，平衡后的公开网页语料负责补足真实使用后的外部表达，BERTopic 则负责把多源文本压缩为可解释、可映射的隐性主题结构。
本轮正式用于 BERTopic 的文本共 648 条，采用平衡后的目标语料口径（asset=targeted_balanced），以降低超长网页对主题结构的单页支配效应。

## 国奖模板式结论

基于国奖风格的图文叙事，第五部分的核心结论可以概括为两条主线。
第一条主线是‘显性治理优先序’：品质/留香、价格/性价比、文化融合、渠道与伴手礼适配共同构成福州本土国潮香氛的显性决策门槛。
第二条主线是‘隐性主题补充证据’：问卷开放题主要指向本土文化整合、茉莉主题开发、礼赠定制和文创协同，公开网页语料则补足了香型、留香、家居香薰与实际使用场景等体验语言。
将二者合并后，第五部分已经能够从‘问题出现了什么’推进到‘问题为什么会阻碍购买、应当如何排序解决’，这也更符合国奖报告中从发现到对策的递进写法。
